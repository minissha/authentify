# Authentify 🔐

A secure document management and verification web application.

## 🚀 Features
- User login & signup
- Upload documents
- Verify documents using hashing
- Simple dashboard

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- PHP
- MySQL

## 📂 Project Structure
- Frontend: HTML/CSS
- Backend: PHP (in /backend)
- Database: MySQL

## ⚠️ Note
This project runs locally using XAMPP.
